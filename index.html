Edit Menu

Go to Blogger.com > Your Blog > Template > Edit HTML and find the following text:- <!-- Top Menu -->

Now change all of the links with your own.

Edit Jumbotron

Go to Blogger.com > Your Blog > Template > Edit HTML and find the following text:- <!-- Jumbotron -->

  • Edit Jumbotron Heading[H2]
  • Edit Jumbotron Paragraph
  • Edit Jumbotron Buttons

    • Now change It.
